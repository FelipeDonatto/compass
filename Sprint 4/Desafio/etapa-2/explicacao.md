É possível reutilizar containers? Em caso positivo, apresente o comando necessário para reiniciar um dos
containers parados em seu ambiente Docker? Não sendo possível reutilizar, justifique sua resposta.

Sim, é possível reutilizar containers.

O comando "docker start (id)" fará com que um container parado seja iniciado.

Da mesma forma, se um container precisar ser reiniciado por algum motivo, o comando "docker restart (nome_do_container)" pode ser utilizado para realizar essa função.
